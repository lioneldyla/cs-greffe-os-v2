# RACI Model

RACI clarifies responsibility across LDGP governance objects.

## Definitions

- Responsible: performs or maintains the work.
- Accountable: owns the final responsibility and decision.
- Consulted: provides input before validation or approval.
- Informed: receives updates after decision or change.

## Compact RACI

| Object | Responsible | Accountable | Consulted | Informed |
|---|---|---|---|---|
| Dataset | Data Steward | Data Owner | Governance Reviewer | Decision Users |
| Document | Knowledge Steward | Document Owner | Document Reviewer | Knowledge Users |
| AI Use Case | AI Governance Steward | Accountable Authority | Human Validator | Affected Users |
| Decision | Decision Reviewer | Accountable Authority | Governance Lead | Action Owners |
| Action | Action Owner | Project Lead | Project Steward | Program Owner |

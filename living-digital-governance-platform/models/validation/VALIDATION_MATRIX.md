# Validation Matrix

| Object | Validation Required | Validator | Enhanced Review | Traceability Required |
|---|---|---|---|---|
| Dataset | Yes | Data Owner | If sensitive or high-impact | Yes |
| Document | Yes | Document Owner | If legal, strategic, or sensitive | Yes |
| Knowledge Asset | Yes | Knowledge Owner | If used for decision support | Yes |
| AI Use Case | Yes | Human Validator | Always before operational use | Yes |
| AI Output | Yes | Human Validator | Always before sensitive use | Yes |
| KPI | Yes | Decision Lead | If used for strategic reporting | Yes |
| Risk | Yes | Risk Owner | If high or critical | Yes |
| Decision | Yes | Accountable Authority | If strategic or sensitive | Yes |
| Action | Yes | Project Lead | If irreversible or external-facing | Yes |
| Project | Yes | Program Lead | If high-impact | Yes |
| Program | Yes | Strategic Lead | If institutional | Yes |

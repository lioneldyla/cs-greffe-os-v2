# CORPUS QUALITY AND METADATA STANDARD V1

## Principes

- Quality before Quantity
- Source before Content
- Transmission before Accumulation
- Community before Repository

## Métadonnées minimales

- source
- date
- auteur ou autorité
- statut temporel
- niveau de confiance
- domaine
- mots-clés
- utilisateurs cibles
- usage prévu
- chemin de transmission

## Niveaux de confiance

| Niveau | Signification |
|---|---|
| A | Texte officiel + pages vérifiées |
| B | Source officielle identifiée |
| C | Support pédagogique vérifié |
| D | Synthèse interne |
| E | Hypothèse ou prospective |
